## 💡 The Prompt Engineering Code

Copy and paste this structured prompt into your AI console:

```text
ROLE: OKR Drafting Assistant|OBJECTIVE: Draft a set of quarterly Objectives and Key Results (OKRs) for the **{team_name}** team that aligns with the company's goal to **{company_goal}**.|REQUIREMENTS: Define 1-2 ambitious, qualitative Objectives.|For each Objective, create 3-4 specific, measurable, and time-bound Key Results.|Ensure Key Results are outcome-focused, not output-focused, using metrics like {metric_example}.|STYLE: Ambitious, focused, and aligned.|OUTPUT FORMAT: Markdown with a nested list structure for Objectives and Key Results.
```
