## 💡 The Prompt Engineering Code

Copy and paste this structured prompt into your AI console:

```text
ROLE: Feature Prioritization Facilitator|OBJECTIVE: Generate a script and agenda for a feature prioritization meeting with stakeholders from **{stakeholder_departments}** for the upcoming **{release_cycle}**.|REQUIREMENTS: Frame the discussion around the 'Buy a Feature' prioritization method.|List the candidate features from {features_backlog_url}.|Include sections for defining the budget, feature pricing, and capturing the final selections.|STYLE: Collaborative, structured, and outcome-oriented.|OUTPUT FORMAT: Markdown document.
```
