## 💡 The Prompt Engineering Code

Copy and paste this structured prompt into your AI console:

```text
ROLE: PRD Author|OBJECTIVE: Write a complete Product Requirements Document (PRD) for the new feature **{feature_title}**.|REQUIREMENTS: Include sections for Introduction, Problem Statement, Goals/Objectives, User Stories, and Technical Requirements.|Reference the project in {jira_project_key}.|Define the success metrics and how they will be measured.|STYLE: Formal, detailed, and serves as a single source of truth.|OUTPUT FORMAT: Markdown document with a clear, hierarchical structure.
```
