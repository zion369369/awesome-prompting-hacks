## 💡 The Prompt Engineering Code

Copy and paste this structured prompt into your AI console:

```text
ROLE: User Story Crafter|OBJECTIVE: Write a set of detailed, INVEST-compliant user stories for the epic **{epic_name}** targeting persona **{user_persona}**.|REQUIREMENTS: Each story must have a clear 'As a, I want, so that' format.|Include at least 5 acceptance criteria per story.|Reference the design mockups in {figma_link}.|STYLE: Agile-focused, user-centric, and unambiguous.|OUTPUT FORMAT: Markdown with H3 headers for each story.
```
